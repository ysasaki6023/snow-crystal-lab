# ❄ 雪結晶ラボ — Snow Crystal Lab

気温・湿度・気圧・水分量を変えると結晶の形が劇的に変わる様子を眺める
3D 雪結晶成長シミュレータ。

## デモ

**https://ysasaki6023.github.io/snow-crystal-lab/** (GitHub Pages)

## ローカル起動

```bash
cd snow-crystal-sim
python3 -m http.server 8741
# → http://localhost:8741 を開く (Chrome / Safari / Edge)
```

ビルド不要・外部 CDN 不要 (three.js は vendor/ に同梱)。

## モデル (なるべく第一原理で、Web で動く程度に簡略化)

Gravner & Griffeath の 3D メソスコピック雪結晶モデル
(*Phys. Rev. E* **79**, 011601, 2009) をベースにした
六方格子 (横6近傍 + 縦2近傍) のセルオートマトン。

- **拡散**: 水蒸気場 d を離散拡散。付着セルは反射境界
- **凍結**: 結晶に接するセルで d → 境界準液層 b へ
- **付着**: b が閾値 β を超えたら結晶化。横 βH / 縦 βV の比が
  板状 ⇔ 柱状の晶癖を決める (温度依存 = 中谷ダイヤグラム)
- **融解**: b の一部が蒸気に戻る (準液層平衡 → ファセット形成)

物理パラメータの接続:

| UI | 物理 | モデルへの効き |
|---|---|---|
| 気温 | 晶癖の異方性 A(T) (中谷ダイヤグラム) | βV/βH 比 |
| 湿度 | 対氷過飽和 → 過剰水蒸気密度 (Magnus式) | 遠方場蒸気密度 ρ |
| 水分量 | 雲水量 (利用可能な水蒸気の上限) | ρ の頭打ち |
| 気圧 | 拡散係数 D ∝ 1/P | 拡散混合率 w |

実装上の工夫:

- **D6h 対称くさび格子**: 結晶の12回対称性を使い 30° の扇形だけを計算
  (12倍高速化 + 完全に対称な結晶)。計算は Web Worker 内
- **枝分かれ**: 純粋な遠方場拡散律速 + キンク閾値 (kinkF) が
  面中央の準液層飽和値より上にあることで、ファセット不安定性
  (面の充填列が面中央で停止 → 角だけが伸びる) が起きる
- 描画は露出面のみの六角柱メッシュ + MeshPhysicalMaterial (transmission)

## 操作

- **プリセット**: 星型樹枝 / 扇形板 / 六角板 / 針 / 角柱 / 低温柱
- **成長中にスライダーを動かせる** — 例: 角柱を1分育ててから
  気温を -15℃ にすると**冠角柱** (柱の両端に板) ができる
- 形態ダイヤグラム上のマーカーが現在の条件と予想形態を表示
- ドラッグで回転、ホイールでズーム、自動回転 ON/OFF
